## 二、每日信息采集模板

### 早晨采集（8:00 自动执行）

```bash
#!/bin/bash
# daily-monitor.sh - 每日热点采集

DATE=$(date +%Y-%m-%d)
REPORT="memory/${DATE}-热点.md"

echo "# ${DATE} 热点监测报告" > $REPORT

# 1. 技术热点 - GitHub
echo "## 技术热点（GitHub）" >> $REPORT
curl -s "https://api.github.com/search/repositories?q=AI+agent&sort=stars&per_page=5" | \
  jq -r '.items[:5] | .[] | "- ⭐\(.stargazers_count) [\( .full_name)](\(.html_url))"' >> $REPORT

# 2. 知乎热榜
echo "## 知乎热榜" >> $REPORT
curl -s "https://www.zhihu.com/api/v3/feed/topstory/hot-lists/total?limit=10" \
  -H "User-Agent: Mozilla/5.0" | \
  jq -r '.data[:5] | .[] | "- \(.target.title)"' >> $REPORT 2>/dev/null || \
  echo "-（获取失败）" >> $REPORT

# 3. 百度热搜（备用方案：用 RSS 订阅）
echo "## 百度热搜" >> $REPORT
echo "- 关注社会情绪关键词：裁员、副业、赚钱、转型" >> $REPORT

# 4. 独立开发者动态
echo "## 独立开发者热点" >> $REPORT
curl -s "https://api.github.com/search/repositories?q=solopreneur+ai&sort=stars&per_page=5" | \
  jq -r '.items[:3] | .[] | "- ⭐\(.stargazers_count) \(.full_name)"' >> $REPORT

# 5. AI 课程/教育热点
echo "## AI 教育热点" >> $REPORT
curl -s "https://api.github.com/search/repositories?q=AI+course&sort=stars&per_page=5" | \
  jq -r '.items[:3] | .[] | "- ⭐\(.stargazers_count) \(.full_name)"' >> $REPORT

echo "✅ 报告已生成: $REPORT"
```

## 三、选题框架（7 天认知系列）

### 第一天：认知唤醒

**目标**：打破认知茧房，让用户意识到 AI 带来的剧变

| 内容方向 | 热点关联 | 标题示例 |
|----------|----------|----------|
| AI 替代危机 | 裁员潮、35岁危机 | "HR 已经开始用 AI 筛简历了，你还在手动投？" |
| 收入格局变化 | 一人公司、Solopreneur | "一个人一年赚 100 万：AI 时代的新可能" |
| 信息差 | AI 焦虑、认知差 | "别人已经用 AI 赚钱了，你还在纠结学不学？" |

**附加信息补充**：
- 真实案例数据（收入变化）
- 行业趋势报告
- 身边人的变化故事

### 第二天：是什么

**目标**：讲清楚 OpenClaw/AI Agent 是什么

| 内容方向 | 热点关联 | 标题示例 |
|----------|----------|----------|
| 概念科普 | MCP、Agent | "MCP 是什么？一文讲清楚 AI 工作的原理" |
| 对比传统 | 编程、工具 | "以前要学 3 个月，现在 3 小时：AI 改变了什么" |
| 实际例子 | 独立开发 | "我用 OpenClaw，一个人做了 5 个人的活" |

### 第三天：为什么

**目标**：为什么现在是最好的时机

| 内容方向 | 热点关联 | 标题示例 |
|----------|----------|----------|
| 成本降低 | API 降价 | "GPT-4 级能力降到 1 分钱：AI 创业门槛没了" |
| 工具成熟 | Cursor、Claude Code | "工具已经ready了，你还在等什么？" |
| 趋势已来 | AI 搜索、vibe coding | "Vibe Coding：不会编程也能做产品的时代到了" |

### 第四天：做什么

**目标**：给出具体可执行的行动方向

| 内容方向 | 热点关联 | 标题示例 |

| 行动清单 | 副业、转型 | "AI 时代独立开发者行动清单：30 天入门路线图" |
| 工具推荐 | MCP、OpenClaw | "独立开发者必备的 5 个 AI 工具" |
| 案例激励 | 成功故事 | "从 0 到 1：会计转型 AI 开发的真实故事" |

### 第五天：怎么做（实操）

**目标**：手把手教学

| 内容方向 | 热点关联 | 标题示例 |
|----------|----------|----------|

| 接入实战 | 微信、飞书 | "微信 + AI：企业微信接入 OpenClaw 实战" |

### 第六天：案例展示

**目标**：用真实案例建立信任

| 内容方向 | 热点关联 | 标题示例 |
|----------|----------|----------|

## 四、热点捕捉机制

### 快速反应模板

当热点出现时（2-4 小时内）：

```
标题：[热点事件] + 观点

开头：发生了什么
分析：大多数人忽略的点是...
案例：类似事件/数据
行动：你可以怎么做

关联：我们课程中关于 xxx 的内容
CTA：扫码了解课程
```

### 热点分类应对

| 热点类型 | 反应策略 | 示例 |
|----------|----------|------|
| **负面危机** | 解决方案导向 | 裁员 → AI 技能自救 |
| **成功案例** | 方法论提取 | 别人赚 100 万 → 他做对了什么 |
| **新工具** | 教程/测评 | 新 MCP 工具 → 怎么用 |
| **行业趋势** | 解读+行动 | AI 搜索冲击 SEO → 怎么办 |
| **社会情绪** | 共鸣+希望 | 35岁焦虑 → 破局思路 |
