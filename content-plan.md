## 🤖 自动化方案

### 目标

自动监测热点 → 生成选题 → 辅助写作 → 发布到公众号

### 方案架构

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  热点监测       │ -> │  选题生成        │ -> │  内容创作       │
│  (定时任务)     │    │  (AI 分析)       │    │  (模板填充)     │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                                              │
         v                                              v
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│  信息源         │    │  选题库         │    │  发布到         │
│  GitHub/HN/Twitter│    │  (Notion/文件)  │    │  GitHub/公众号  │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

### 1. 热点监测（每日自动）

创建定时任务脚本 `monitor-hot.sh`：

```bash
#!/bin/bash
# 热点监测脚本 - 每天早上 8 点运行

echo "=== $(date) 热点监测 ==="

# 1. GitHub trending
curl -s "https://api.github.com/search/repositories?q=openclaw&sort=stars" | \
  jq '.items[:5] | .[] | "⭐ \(.stargazers_count) \(.full_name)"'

# 2. 关键词搜索量
curl -s "https://trends.google.com/trends/explore?q=openclaw,AI+agent" | \
  grep -oP 'title.*?content' | head -5

# 3. 输出报告
echo "=== 今日热点 ===" >> memory/$(date +%Y-%m-%d).md
echo "## 热点监测" >> memory/$(date +%Y-%m-%d).md
```

### 2. 选题生成（AI 驱动）

根据热点自动生成选题：

```bash
#!/bin/bash
# 选题生成脚本

# 读取今日热点
HOT_TOPICS=$(cat memory/$(date +%Y-%m-%d).md)

# 调用 AI 生成选题
openclaw --prompt "根据今日热点生成 3 个适合公众号的选题：
$HOT_TOPICS

要求：
- 面向独立开发者
- 结合 OpenClaw 课程推广
- 标题要有吸引力
- 输出 JSON 格式"
```

### 3. 内容模板

创建常用模板：

| 模板名 | 用途 | 变量 |
|--------|------|------|
| `教程-基础.md` | 入门教程 | 工具名、步骤 |
| `教程-进阶.md` | 进阶教程 | 技能名、代码 |
| `观点-热点.md` | 热点评论 | 事件、观点 |
| `案例-故事.md` | 学员案例 | 人物、结果 |
| `清单-工具.md` | 工具清单 | 工具列表 |

### 4. 发布流程

```bash
# 1. 撰写文章保存到 drafts
# 2. 同步到 GitHub
git add drafts/$(date +%Y-%m-%d)-*.md
git commit -m "content: 添加 $(date +%Y-%m-%d) 文章草稿"
git push origin main

# 3. 可选：推送到公众号（需要配置）
# 调用微信 API 创建草稿
curl -X POST "https://api.weixin.qq.com/cgi-bin/draft/add?access_token=$TOKEN" \
  -d @- <<EOF
{
  "articles": [{
    "title": "文章标题",
    "author": "作者",
    "content": "文章内容（HTML）",
    "digest": "摘要"
  }]
}
EOF
```

### 5. 定时任务配置

```bash
# crontab -e
# 每天早上 8 点监测热点
0 8 * * * /path/to/monitor-hot.sh

# 每天下午 2 点生成选题
0 14 * * * /path/to/generate-topics.sh
```
