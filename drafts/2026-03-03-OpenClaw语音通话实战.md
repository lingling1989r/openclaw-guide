## 一、什么是 OpenClaw 语音通话？

OpenClaw 语音通话（Voice Call）是 OpenClaw 的一个**插件**，支持：

- 📞 **外呼通知** — 让 AI 主动给你打电话
- 📞 **双向对话** — 接听来电并与 AI 对话
- 🌐 **多服务商支持** — Twilio、Telnyx、Plivo

> **数据来源**：GitHub Issues 显示，语音通话功能是 OpenClaw 社区**需求最高**的功能之一，议题讨论有 **39+ 条评论**。
> 
> 参考：https://github.com/openclaw/openclaw/issues?q=is%3Aissue+voice-call

## 二、准备工作

在开始之前，你需要准备：

### 2.1 一个 OpenClaw 已部署的环境

确保你的 OpenClaw 已经正常运行。如果没有，请先参考部署教程。

### 2.2 电话服务商账号（选一个）

| 服务商 | 价格 | 特点 | 官网 |
|--------|------|------|------|
| **Twilio** | $1/100分钟 | 最流行，文档全 | twilio.com |
| **Telnyx** | $0.005/分钟 | 性价比高 | telnyx.com |
| **Plivo** | $0.015/分钟 | 支持 XML | plivo.com |

**推荐新手从 Twilio 开始**，中文资料最多。

### 2.3 TTS 语音合成

你需要配置 TTS（文字转语音），支持：

- **OpenAI TTS** — 免费额度多
- **ElevenLabs** — 音质更好，支持中文

## 三、安装语音插件

### 3.1 安装插件

官方文档来源：https://docs.openclaw.ai/plugins/voice-call

```bash
# 方式一：从 npm 安装（推荐）
openclaw plugins install @openclaw/voice-call

# 方式二：本地安装（开发用）
openclaw plugins install ./extensions/voice-call
```

### 3.2 重启 Gateway

安装完成后，必须重启 Gateway 才能加载插件：

```bash
openclaw gateway restart
```

## 四、配置电话服务商

### 4.1 Twilio 配置（推荐）

1. **注册 Twilio 账号**：https://www.twilio.com
2. **获取 Credentials**：在 Console 中找到：
   - Account SID
   - Auth Token
   - 一个美国电话号码（From Number）

3. **配置 OpenClaw**：

在 `~/.openclaw/openclaw.json` 中添加：

```json5
{
  plugins: {
    entries: {
      "voice-call": {
        enabled: true,
        config: {
          provider: "twilio",
          fromNumber: "+15550001234",  // 你的 Twilio 电话号码
          
          twilio: {
            accountSid: "ACxxxxxxxx",
            authToken: "你的Auth Token",
          },

          // webhook 配置（需要公网访问）
          serve: {
            port: 3334,
            path: "/voice/webhook",
          },

          // 外呼模式
          outbound: {
            defaultMode: "notify",  // notify | conversation
          },
        },
      },
    },
  },
}
```

> **来源**：OpenClaw 官方文档
> 参考：https://docs.openclaw.ai/plugins/voice-call#config

### 4.2 配置 TTS 语音

在同一个配置文件中添加：

```json5
{
  messages: {
    tts: {
      provider: "openai",
      openai: {
        voice: "alloy",  // 可选：alloy, echo, fable, onyx, nova, shimmer
        model: "gpt-4o-mini-tts",
      },
    },
  },
}
```

或者使用 ElevenLabs（音质更好）：

```json5
{
  plugins: {
    entries: {
      "voice-call": {
        config: {
          tts: {
            provider: "elevenlabs",
            elevenlabs: {
              apiKey: "你的ElevenLabs Key",
              voiceId: "pMsXgVXv3BLzUgSXRplE",
              modelId: "eleven_multilingual_v2",
            },
          },
        },
      },
    },
  },
}
```

> **来源**：OpenClaw 官方文档 — TTS for calls
> 参考：https://docs.openclaw.ai/plugins/voice-call#tts-for-calls

### 4.3 公网访问配置

Twilio 需要一个**公网可访问**的 webhook URL。有三种方案：

#### 方案 A：Tailscale Funnel（推荐，免费）

```json5
{
  plugins: {
    entries: {
      "voice-call": {
        config: {
          tunnel: {
            provider: "tailscale",
            mode: "funnel",
            path: "/voice/webhook",
          },
        },
      },
    },
  },
}
```

#### 方案 B：ngrok

```json5
{
  plugins: {
    entries: {
      "voice-call": {
        config: {
          publicUrl: "https://你的ngrok.io/voice/webhook",
        },
      },
    },
  },
}
```

#### 方案 C：固定域名（生产环境）

```json5
{
  plugins: {
    entries: {
      "voice-call": {
        config: {
          publicUrl: "https://voice.example.com/voice/webhook",
          webhookSecurity: {
            allowedHosts: ["voice.example.com"],
          },
        },
      },
    },
  },
}
```

## 五、实战：打出第一通 AI 电话

### 5.1 使用 CLI 拨打

```bash
# 通知模式（单向，只读通知）
openclaw voicecall --to +15550005678 --message "你好，这是 OpenClaw AI 电话测试"

# 对话模式（双向，可交互）
openclaw voicecall --to +15550005678 --mode conversation
```

### 5.2 使用 Tool 调用

在代码中调用 `voice_call` 工具：

```json
{
  "name": "voice_call",
  "arguments": {
    "to": "+15550005678",
    "message": "你好！有什么可以帮你的？"
  }
}
```

### 5.3 配置Inbound（接听电话）

如果你想让别人也能打给你，配置 inbound：

```json5
{
  plugins: {
    entries: {
      "voice-call": {
        config: {
          inboundPolicy: "allowlist",
          allowFrom: ["+15550001234"],  // 白名单号码
          inboundGreeting: "你好！我是 OpenClaw AI，请问有什么可以帮您？",
        },
      },
    },
  },
}
```

> **来源**：OpenClaw 官方文档 — Inbound calls
> 参考：https://docs.openclaw.ai/plugins/voice-call#inbound-calls

## 六、常见问题与排查

### Q1: 打不出去，提示 "provider not configured"？

检查 `~/.openclaw/openclaw.json` 中的 `plugins.entries.voice-call.config` 是否正确配置。

### Q2: Twilio webhook 验证失败？

- 确保使用 ngrok 固定 URL 或 Tailscale Funnel
- 检查 `publicUrl` 是否与实际公网地址一致
- 参考：https://docs.openclaw.ai/plugins/voice-call#webhook-security

### Q3: 语音听起来很奇怪？

- 尝试换 TTS 提供商（ElevenLabs 效果更好）
- 检查 `voice` 参数是否适合中文

### Q4: 如何限制通话时长？

```json5
{
  plugins: {
    entries: {
      "voice-call": {
        config: {
          maxDurationSeconds: 300,  // 最长 5 分钟
          staleCallReaperSeconds: 360,  // 6 分钟后自动挂断
        },
      },
    },
  },
}
```

> **来源**：OpenClaw 官方文档 — Stale call reaper
> 参考：https://docs.openclaw.ai/plugins/voice-call#stale-call-reaper
